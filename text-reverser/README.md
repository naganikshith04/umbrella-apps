# Text Reverser

Reverse text, words, or letters with multiple reversal pattern options

## Features

- Reverse entire text (mirror text)
- Reverse word order but keep letters normal
- Reverse each word individually
- Flip text upside down (Unicode transformation)
- Reverse by sentence or paragraph

## Keywords

reverse text, backwards text, text reverser, mirror text, flip text, reverse words, backwards generator, text flipper

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
