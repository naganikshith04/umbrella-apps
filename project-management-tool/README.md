# Project Management Tool

Project Management Tool with kanban boards, task tracking, and team collaboration

## Features
- Kanban board with drag-and-drop
- Task cards with details, priorities, due dates
- Multiple project views
- Task filtering and sorting
- Export to JSON/CSV
- LocalStorage persistence

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd project-management-tool
vercel --prod
```
