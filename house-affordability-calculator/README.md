# House Affordability Calculator

Calculate how much house you can afford based on income, debts, down payment, and current rates

## Features

- Income-based affordability (28/36 rule)
- Down payment and closing cost estimator
- Current mortgage rate integration
- Property tax and insurance estimates by location
- Monthly budget breakdown visualization
