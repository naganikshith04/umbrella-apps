# VO2 Max Calculator

Estimate cardiovascular fitness level using simple field tests

## Features

- Multiple test methods (Cooper test, Rockport walk, step test)
- Age and gender-adjusted fitness ratings
- Fitness level categories (poor to excellent)
- Training recommendations for improvement
- Comparison to population averages

## Keywords

vo2 max calculator, cardiovascular fitness calculator, aerobic capacity calculator, cooper test calculator, fitness level test
