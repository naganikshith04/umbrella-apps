# Image Format Converter

Convert images between different formats

## Features
- Convert between JPG, PNG, WebP, GIF
- Batch conversion support
- Maintain or adjust quality
- Preview converted images
- Download as ZIP

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd image-format-converter
vercel --prod
```
