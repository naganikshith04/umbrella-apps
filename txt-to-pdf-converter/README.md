# TXT to PDF Converter

Convert plain text files to formatted PDF documents with customization options

## Features

- Font family and size selection
- Page margins and orientation
- Line spacing adjustment
- Syntax highlighting for code
- Automatic page numbering

## Keywords

txt to pdf, text to pdf, convert text file to pdf, plain text to pdf, txt pdf converter

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
