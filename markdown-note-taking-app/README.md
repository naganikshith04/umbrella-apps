# Markdown Note Taking App

Markdown-based Note Taking App with folders, tags, search, and export

## Features
- Folder/notebook organization
- Real-time markdown preview
- Tag system
- Search across notes
- Export to HTML/PDF
- LocalStorage persistence
- Keyboard shortcuts

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd markdown-note-taking-app
vercel --prod
```
