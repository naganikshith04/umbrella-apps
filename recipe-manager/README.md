# Recipe Manager

Recipe Manager with ingredients, instructions, meal planning, and shopping lists

## Features
- Recipe cards with images
- Ingredient list management
- Step-by-step instructions
- Meal planning calendar
- Shopping list generation
- Recipe search and filter
- Export shopping list

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd recipe-manager
vercel --prod
```
