# Intermittent Fasting Calculator

Calculate eating windows and fasting schedules for various IF protocols

## Features

- Multiple IF protocols (16:8, 18:6, 20:4, 5:2, OMAD)
- Personalized eating window calculator
- Meal timing optimizer
- Calorie distribution across eating window
- Fasting timer with notifications
