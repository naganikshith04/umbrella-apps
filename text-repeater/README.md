# Text Repeater

Repeat text, words, or characters a specified number of times with separator options

## Features

- Repeat text 1-10000 times
- Custom separator between repetitions
- New line, space, or custom delimiter
- Add prefix and suffix to each repetition
- Number each repetition optionally

## Keywords

text repeater, repeat text, duplicate text, text multiplier, copy text multiple times, text duplicator, repeat words, text cloner

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
