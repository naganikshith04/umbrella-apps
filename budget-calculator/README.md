# Budget Calculator

Create a monthly budget using 50/30/20 rule or custom allocation with expense tracking and recommendations

## Features

- Income and expense category inputs
- 50/30/20 rule automatic allocation (needs/wants/savings)
- Custom budget category creation
- Overspending alerts and recommendations
- Monthly vs actual comparison

## Keywords

budget calculator, monthly budget calculator, budgeting calculator, 50/30/20 budget calculator, personal budget calculator, household budget calculator, expense calculator, budget planner calculator
