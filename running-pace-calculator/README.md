# Running Pace Calculator

Calculate running pace, finish time, and training paces for race preparation

## Features

- Pace/time/distance calculator (solve for any variable)
- Race finish time predictor
- Training pace calculator (easy, tempo, interval)
- Split time calculator for mile/km markers
- Pace conversion (min/mile to min/km)
