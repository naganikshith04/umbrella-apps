# JSON to CSV Converter

Convert JSON to CSV and vice versa

## Features
- Paste or upload JSON
- Flatten nested JSON
- Preview as table
- Convert to CSV
- Download result

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd json-csv-converter
vercel --prod
```
