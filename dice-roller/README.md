# Dice Roller

Roll dice with custom sides for games and random decisions

## Features

- number of dice selector
- sides selector (4, 6, 8, 10, 12, 20)
- roll button
- show results
- roll history
- total sum

## Keywords

dice roller, roll dice, virtual dice, random dice

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```
