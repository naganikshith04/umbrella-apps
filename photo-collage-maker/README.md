# Photo Collage Maker

Create photo collages with multiple templates

## Features
- Upload multiple photos
- Choose from templates
- Drag and position photos
- Add text overlays
- Download final collage

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd photo-collage-maker
vercel --prod
```
