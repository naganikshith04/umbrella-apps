# Text Difference Checker

Compare two texts side-by-side and highlight additions, deletions, and changes

## Features

- Side-by-side or inline diff view
- Color-coded additions, deletions, and modifications
- Word-level and character-level comparison modes
- Ignore whitespace and case options
- Export diff report as HTML or plain text
