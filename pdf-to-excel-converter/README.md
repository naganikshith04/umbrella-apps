# PDF to Excel Converter

Extract tables from PDF files into editable Excel spreadsheets with accuracy

## Features

- Table detection and extraction
- Multi-page PDF support
- Preserves cell formatting
- OCR for scanned PDFs (premium)
- Column alignment optimization

## Keywords

pdf to excel, pdf to xlsx, convert pdf to spreadsheet, pdf table extractor, pdf to xls online

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
