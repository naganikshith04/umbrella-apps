# Image Compressor Pro

Compress images without losing quality with advanced options

## Features
- Upload multiple images (drag & drop)
- Adjust compression quality with slider
- Preview before/after comparison
- Download individual or batch as ZIP
- Support JPG, PNG, WebP formats
- Show file size reduction statistics

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd image-compressor
vercel --prod
```
