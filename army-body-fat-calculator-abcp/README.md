# Army Body Fat Calculator (ABCP)

Calculate body fat percentage using official US Army standards and tape test method

## Features

- Gender-specific Army tape test calculator
- Pass/fail status with maximum allowable percentages
- Age-based standards chart
- Height/weight screening table
- Improvement recommendations for failing scores

## Keywords
