# Time Zone Converter

Convert time between different time zones with daylight saving awareness

## Features

- Multi-timezone comparison (up to 6 zones)
- Automatic DST adjustment
- Meeting time finder for multiple zones
- City search with popular locations
- URL sharing for specific time conversions
