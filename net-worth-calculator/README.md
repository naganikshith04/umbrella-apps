# Net Worth Calculator

Calculate and track your total net worth with asset and liability breakdown and percentile ranking

## Features

- Asset categories (cash, investments, real estate, vehicles, etc.)
- Liability categories (mortgages, loans, credit cards, etc.)
- Net worth percentile by age comparison
- Historical tracking and growth visualization
- Net worth goal setting

## Keywords

net worth calculator, calculate net worth, net worth tracker, personal net worth calculator, what is my net worth, net worth by age, wealth calculator, assets minus liabilities calculator

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
