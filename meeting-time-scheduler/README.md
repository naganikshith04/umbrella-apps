# Meeting Time Scheduler

Find optimal meeting times across multiple time zones for global teams

## Features

- Visual availability grid for all zones
- Business hours highlighting
- Best time suggestions algorithm
- Calendar integration (Google, Outlook)
- Generate calendar invites with all zones
