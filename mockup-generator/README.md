# Mockup Generator

Create device mockups

## Features
- Upload screenshot
- Choose device template
- Adjust position and scale
- Add background
- Download mockup image

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd mockup-generator
vercel --prod
```

## Tech
- HTML/CSS/JavaScript
- External libraries: html2canvas, file-saver
- AdSense integrated
- Fully responsive
