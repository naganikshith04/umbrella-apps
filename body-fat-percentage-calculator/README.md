# Body Fat Percentage Calculator

Estimate body fat using multiple measurement methods including Navy, caliper, and visual

## Features

- US Navy method (circumference measurements)
- 3-site and 7-site skinfold caliper methods
- Visual estimation guide with reference images
- Body fat category classification
- Lean body mass calculation

## Keywords

body fat calculator, body fat percentage, navy body fat calculator, caliper calculator, body composition calculator, lean body mass
