# Certificate Generator

Create custom certificates

## Features
- Choose template
- Customize text fields
- Add signatures
- Upload logos
- Preview before download
- Batch generation
- Export as PDF

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd certificate-generator
vercel --prod
```
