# Tip Calculator

Calculate tips and split bills easily for restaurants

## Features

- bill amount input
- tip percentage slider
- split count
- calculate total per person
- round up option

## Keywords

tip calculator, gratuity calculator, bill splitter, restaurant tip

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
