# Add/Subtract Days Calculator

Add or subtract days, weeks, months, or years from any date

## Features

- Add/subtract multiple time units simultaneously
- Business days only mode
- Recurring date calculator (every X days)
- Batch date calculation (multiple operations)
- Calendar view of result

## Keywords

add days to date, date calculator, days from today, subtract days, date plus days, 90 days from today

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```
