# Student Loan Calculator

Calculate student loan payments with federal and private loan options, forgiveness programs, and refinancing scenarios

## Features

- Federal loan payment plans (Standard, PAYE, IBR, REPAYE)
- Public Service Loan Forgiveness calculator
- Refinancing comparison tool
- Grace period and deferment impact
- Parent PLUS loan calculations

## Keywords

student loan calculator, student loan payment calculator, student debt calculator, loan forgiveness calculator, student loan repayment calculator, income driven repayment calculator, pslf calculator, student loan refinance calculator
