# Watermark Tool

Add watermarks to images

## Features
- Upload images
- Add text or image watermark
- Adjust position and opacity
- Batch watermarking
- Download watermarked images

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd watermark-tool
vercel --prod
```

## Tech
- HTML/CSS/JavaScript
- External libraries: html2canvas, jszip, file-saver
