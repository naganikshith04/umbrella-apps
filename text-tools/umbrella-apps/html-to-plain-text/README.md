# HTML to Plain Text

Strip HTML tags and convert formatted HTML to clean plain text

## Features

- Remove all HTML tags completely
- Preserve line breaks and paragraph structure
- Convert HTML entities to characters
- Remove inline styles and scripts
- Option to keep links as plain URLs

## Keywords

html to text, remove html tags, strip html, html stripper, convert html to text, html tag remover, clean html, extract text from html

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
