# Sentence Counter

Count sentences and analyze sentence structure with average length statistics

## Features

- Accurate sentence counting (handles abbreviations)
- Average sentence length in words
- Longest and shortest sentence identification
- Sentence complexity score
- Readability grade level

This app is ready to deploy to Vercel:

