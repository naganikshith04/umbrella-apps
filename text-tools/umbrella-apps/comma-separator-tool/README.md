# Comma Separator Tool

Add or remove commas between items and convert lists to comma-separated format

## Features

- Convert line-separated list to comma-separated
- Remove commas and split to lines
- Add quotes around each item
- Custom separator (comma, semicolon, pipe)
- Remove duplicates while converting

## Keywords

comma separator, add commas, comma separated, list to comma, csv formatter, comma delimiter, separate with commas, comma tool
