# Whitespace Remover

Remove all spaces, tabs, and line breaks or selectively clean whitespace

## Features

- Remove all whitespace completely
- Remove only extra spaces (keep single)
- Remove line breaks only
- Trim start and end whitespace
- Convert tabs to spaces or vice versa

## Keywords

remove spaces, whitespace remover, remove line breaks, delete spaces, strip whitespace, remove tabs, space remover, text compressor

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```
