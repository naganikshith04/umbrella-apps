# Text Splitter

Split text by delimiter, character count, word count, or custom patterns

## Features

- Split by comma, space, newline, or custom delimiter
- Split into chunks of N characters or words
- Regex pattern splitting
- Preserve or remove delimiter option
- Number each split section

## Keywords

text splitter, split text, text divider, split by comma, text separator, divide text, split string, text chunker

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
