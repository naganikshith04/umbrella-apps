# Duplicate Line Remover

Remove duplicate lines from text while preserving original order or sorting

## Features

- Remove exact duplicate lines
- Case-sensitive or case-insensitive matching
- Keep first or last occurrence option
- Sort output alphabetically (optional)
- Show count of removed duplicates

## Keywords
