# Invisible Character Detector

Find and highlight invisible Unicode characters, zero-width spaces, and hidden text

## Features

- Detect zero-width spaces and joiners
- Highlight invisible Unicode characters
- Show character codes and names
- Remove all invisible characters option
- Visualize whitespace characters
