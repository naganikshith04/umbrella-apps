# Emergency Fund Calculator

Calculate how much emergency savings you need based on monthly expenses and risk factors

## Features

- Monthly expense input with category breakdown
- Job security and income stability assessment
- Recommended months of expenses (3-12 months)
- Savings timeline to reach goal
- Risk factor adjustments (dependents, homeowner, etc.)
