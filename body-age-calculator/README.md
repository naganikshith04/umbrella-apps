# Body Age Calculator

Calculate biological age based on fitness metrics and lifestyle factors

## Features

- Fitness test-based age calculation (VO2 max, flexibility, strength)
- Lifestyle factor assessment (sleep, stress, diet)
- Health marker inputs (blood pressure, cholesterol)
- Biological vs chronological age comparison
- Improvement recommendations for younger body age

## Keywords

body age calculator, biological age calculator, fitness age calculator, real age calculator, health age calculator

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.
