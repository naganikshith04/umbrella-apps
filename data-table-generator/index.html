How to Use

  1. Import: Click "Browse Files" or drag & drop your CSV/Excel file into the upload zone
  2. Sort: Click any column header to sort data (click again to reverse order)
  3. Search: Use the search box to find data across all columns
  4. Filter: Select a column and enter a value to filter specific data
  5. Navigate: Use pagination controls to browse through large datasets
  6. Export: Download your filtered/sorted data as CSV or Excel
  7. Print: Generate a print-friendly view of your table
