# Time Card Calculator

Calculate weekly time card totals with multiple punch-in/out entries

## Features

- Multi-day time entry grid
- Automatic break deduction
- Weekly total with overtime
- Hourly rate wage calculator
- Print-friendly timesheet format

## Keywords

time card calculator, timecard calculator, punch clock calculator, weekly hours calculator, time sheet calculator

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```
