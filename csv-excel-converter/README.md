# CSV to Excel Converter

Convert between CSV and Excel formats

## Features
- Upload CSV file
- Preview data in table
- Edit data inline
- Convert to Excel (.xlsx)
- Download converted file
- Batch conversion

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd csv-excel-converter
vercel --prod
```
