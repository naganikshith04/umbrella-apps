# Text to Speech

Convert written text to natural-sounding speech with voice and speed controls

## Features

- Multiple language support (20+ languages)
- Male and female voice options
- Adjustable speech rate and pitch
- Download as MP3 audio file
- Highlight text as it's being read

## Keywords

text to speech, tts, text reader, speech generator, text to audio, voice generator, read text aloud, text to mp3

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
