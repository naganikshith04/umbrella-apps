# Slug Generator

Convert titles and text into URL-friendly slugs with SEO optimization

## Features

- Convert to lowercase with hyphens
- Remove special characters and accents
- Transliterate non-Latin characters
- Custom separator (hyphen, underscore)
- Maximum length limiting with smart truncation

This app is ready to deploy to Vercel:

```bash

```

```bash

```
