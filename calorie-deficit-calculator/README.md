# Calorie Deficit Calculator

Calculate safe calorie deficit and predict weight loss timeline

## Features

- TDEE-based deficit calculation
- Safe deficit limits (max 1000 cal/day)
- Weekly and monthly weight loss projections
- Goal date calculator (reach target weight by date)
- Plateau prevention tips
