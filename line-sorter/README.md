# Line Sorter

Sort text lines alphabetically, numerically, by length, or reverse order

## Features

- Alphabetical (A-Z or Z-A) sorting
- Numerical sorting with natural number handling
- Sort by line length (shortest/longest first)
- Case-sensitive or case-insensitive options
- Random shuffle mode
