# Mind Map Creator

Create visual mind maps

## Features
- Add central idea
- Add branches and sub-branches
- Customize colors
- Drag to rearrange
- Export as image
- Save to local storage

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd mind-map-creator
vercel --prod
```
