# PowerPoint to PDF Converter

Convert presentations to PDF while maintaining animations and slide notes

## Features

- Converts PPT and PPTX files
- Include or exclude speaker notes
- Handout layout options
- Custom slides per page
- Maintains transitions as static slides

## Keywords

powerpoint to pdf, ppt to pdf, convert presentation to pdf, pptx to pdf online, slides to pdf

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
