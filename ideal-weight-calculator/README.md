# Ideal Weight Calculator

Calculate ideal body weight using multiple medical formulas and body frame size

## Features

- Multiple formulas (Robinson, Miller, Devine, Hamwi)
- Body frame size calculator (wrist measurement)
- Healthy weight range display
- BMI correlation comparison
- Age and gender-specific recommendations

## Keywords

ideal weight calculator, healthy weight calculator, target weight calculator, perfect weight calculator, ideal body weight formula
