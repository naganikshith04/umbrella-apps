# Umbrella Site - Main Landing Page

## Overview
This is the main landing page that links to all 45 generated apps.

## Structure
- index.html - Main landing page
- css/umbrella.css - Styles

## Features
- ✅ Groups apps by category
- ✅ Beautiful, responsive design
- ✅ AdSense integrated (ca-pub-9048718254553458)
- ✅ SEO optimized
- ✅ Links to all 45 apps

## Deployment

### Option 1: Deploy as Main Site
```bash
cd generated-apps/umbrella-site
vercel --prod
```

### Option 2: Add to Umbrella Apps Repo
```bash
cp -r generated-apps/umbrella-site/* ~/umbrella-apps/
cd ~/umbrella-apps
git add .
git commit -m "Add umbrella site"
git push
```
