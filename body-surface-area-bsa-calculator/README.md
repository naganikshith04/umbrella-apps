# Body Surface Area (BSA) Calculator

Calculate body surface area for medical dosing and health assessments

## Features

- Multiple BSA formulas (Mosteller, DuBois, Haycock)
- Metric and imperial units
- Medical dosing reference guide
- Burn assessment percentage tool
- Pediatric-specific calculations
