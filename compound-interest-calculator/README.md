# Compound Interest Calculator

Calculate investment growth with compound interest including regular contributions and time visualization

## Features

- Compound interest with multiple compounding frequencies (daily, monthly, quarterly, annually)
- Regular contribution options (monthly, quarterly, yearly)
- Visual growth chart showing contribution vs interest earned
- Side-by-side comparison of different scenarios
- Inflation adjustment option
