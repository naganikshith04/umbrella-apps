# Markdown to HTML Converter

Convert Markdown to styled HTML

## Features
- Write or paste Markdown
- Live preview
- Syntax highlighting
- Choose HTML template
- Export as HTML file
- Copy to clipboard

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd markdown-html-converter
vercel --prod
```

## Tech
- HTML/CSS/JavaScript
- External libraries: prismjs, file-saver
- AdSense integrated
- Fully responsive
