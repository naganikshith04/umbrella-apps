# Portfolio Generator

Portfolio Generator with projects, skills, testimonials, and contact form

## Features
- Project showcase
- Skills section with proficiency
- Testimonials carousel
- Contact form
- About me section
- Export as HTML
- Multiple themes

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd portfolio-generator
vercel --prod
```

- HTML/CSS/JavaScript

- AdSense integrated
- Fully responsive
