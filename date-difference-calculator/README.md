# Date Difference Calculator

Calculate the exact duration between two dates in multiple formats

## Features

- Duration in years, months, weeks, days
- Business days calculator (excluding weekends)
- Include/exclude start and end dates option
- Working hours calculation
- Export results as text or PDF

## Keywords

date difference calculator, days between dates, calculate days, date duration, time between dates, business days calculator

This app is ready to deploy to Vercel:

```bash

```
