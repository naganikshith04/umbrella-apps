# Refinance Calculator

Calculate if refinancing your mortgage makes financial sense with break-even analysis and lifetime savings

## Features

- Current vs new loan comparison
- Break-even point calculation (months to recoup closing costs)
- Lifetime interest savings
- Cash-out refinance scenarios
- ARM to fixed rate conversion analysis
