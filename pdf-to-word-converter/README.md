# PDF to Word Converter

Convert PDF files to editable Word documents while preserving formatting and layout

## Features

- Batch conversion support (up to 20 files)
- OCR for scanned PDFs (premium feature)
- Preserves images, tables, and formatting
- Download as DOCX or DOC format
- Password-protected PDF support

## Keywords

pdf to word, pdf to docx converter, convert pdf to word, pdf to doc, pdf converter online

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
