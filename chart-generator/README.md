# Chart Generator Pro

Create beautiful charts from your data

## Features
- Upload CSV or paste data
- Choose chart type (bar, line, pie, scatter)
- Customize colors and labels
- Interactive tooltips
- Download as image
- Export data as JSON

## Files
- `index.html` - Complete HTML
- `css/styles.css` - Styling
- `js/app.js` - Functionality

## Deployment
```bash
cd chart-generator
vercel --prod
```

## Tech
- HTML/CSS/JavaScript
- External libraries: chart.js, papaparse, html2canvas, file-saver
- AdSense integrated
- Fully responsive
