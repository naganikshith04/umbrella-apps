# Heart Rate Zone Calculator

Calculate personalized heart rate training zones for optimal cardio workouts

## Features

- Maximum heart rate calculation (220-age and Karvonen methods)
- 5-zone heart rate chart (recovery, fat burn, cardio, threshold, peak)
- Zone-specific training benefits
- BPM ranges for each zone
- Workout intensity recommendations

## Keywords

heart rate zone calculator, target heart rate calculator, cardio zone calculator, max heart rate calculator, fat burning heart rate
