# HEIC to JPG Converter

Convert iPhone HEIC photos to widely compatible JPG format instantly

## Features

- Drag-and-drop batch conversion
- Maintains photo quality
- Preserves EXIF metadata
- Live Photos frame extraction
- ZIP download for multiple files
