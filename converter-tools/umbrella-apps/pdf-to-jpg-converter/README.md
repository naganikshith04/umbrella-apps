# PDF to JPG Converter

Extract pages from PDF documents as high-quality JPG images

## Features

- Select specific pages or all pages
- DPI/quality adjustment
- Batch PDF processing
- ZIP download for multiple images
- Page range selection

## Keywords

pdf to jpg, pdf to image, convert pdf to jpeg, pdf to png, pdf image extractor

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
