# Video to MP4 Converter

Convert any video format to MP4 for universal playback compatibility

## Features

- Supports AVI, MOV, WMV, MKV, FLV input
- Resolution presets (1080p, 720p, 480p)
- Codec selection (H.264, H.265)
- Audio bitrate adjustment
- Fast conversion with hardware acceleration
