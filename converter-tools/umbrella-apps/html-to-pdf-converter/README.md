# HTML to PDF Converter

Convert web pages and HTML files to PDF documents with styling preserved

## Features

- URL to PDF conversion
- HTML file upload support
- Custom page size and margins
- Header and footer customization
- JavaScript rendering support

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
