# PNG to ICO Converter

Create favicon ICO files from PNG images with multiple size variants

## Features

- Generates multi-resolution ICO files
- Supports 16x16 to 256x256 sizes
- Transparent background support
- Preview all icon sizes
- Batch favicon creation

This app is ready to deploy to Vercel:

```bash

```
