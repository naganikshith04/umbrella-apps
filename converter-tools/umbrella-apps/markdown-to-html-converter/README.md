# Markdown to HTML Converter

Convert Markdown formatted text to clean HTML with live preview

## Features

- Real-time Markdown to HTML conversion
- Live preview pane with styling
- Support for GitHub Flavored Markdown
- Syntax highlighting for code blocks
- Copy HTML or download as file

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
