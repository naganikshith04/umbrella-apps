# CSV to Excel Converter

Transform CSV files into formatted Excel spreadsheets with data type detection

## Features

- Auto-detect data types
- Custom delimiter support
- Header row detection
- Multiple sheets from multiple CSVs
- Basic formatting options

## Keywords

csv to excel, convert csv to xlsx, csv to xls, csv excel converter, csv spreadsheet online

## Deployment

This app is ready to deploy to Vercel:

```bash
vercel deploy
```

Or simply push to GitHub and connect to Vercel for automatic deployments.

## Local Development

Open `index.html` in a web browser or use a local server:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000
